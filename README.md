## Features 🌟
- [x] Music lists
- [x] Parsing music metadata
- [x] Bottom player
- [x] Playback controls
- [x] Playback modes
- [x] Responsive
- [x] System Media Controls
- [ ] WebDAV support
- [ ] Cross-platform client
- [ ] Music Grouping
- [ ] Music Search
- [ ] Configuration Sync
- [ ] Fullscreen player
