## 功能 🌟
- [x] 音乐列表
- [x] 解析音乐元数据
- [x] 底部播放器
- [x] 播放控制
- [x] 播放模式
- [x] 响应式
- [x] 系统媒体控制
- [ ] WebDAV 支持
- [ ] 跨平台客户端
- [ ] 音乐分组
- [ ] 音乐搜索
- [ ] 配置同步
- [ ] 全屏播放器
